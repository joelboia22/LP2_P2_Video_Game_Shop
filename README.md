# Sistema Escolar Boletim - SEB

> O SEB será uma plataforma desenvolvida com o framework Spring Boot e o sistema de gestão de base de dados MySQL, sendo o objetivo principal da aplicação disponibilizar o cadastro de alunos, professores e disciplinas, além de armazenar e realizar os cálculos das notas dos respectivos discentes do Instituto Municipal de Arapiraca - IMA (instituição fictícia).
